# Employee Management System

A desktop application built in **C#** (.NET) for managing employee records within an organization.

---

## Overview

This project provides a straightforward interface for performing common employee administration tasks, including creating, viewing, updating, and removing employee records.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Language | C# |
| Framework | .NET (Windows Forms / WPF) |
| IDE | Visual Studio |
| Project File | `Employee Management.sln` |

---

## Features

- **Add Employees** — Create new employee records with relevant details
- **View Employees** — Browse and search existing employee records
- **Update Employees** — Edit and save changes to employee information
- **Delete Employees** — Remove employee records from the system

---

## Getting Started

### Prerequisites

- [Visual Studio](https://visualstudio.microsoft.com/) (2019 or later recommended)
- .NET Framework / .NET SDK (version depends on project target)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/thejazzynator/Employee-Management.git
   ```

2. **Open the solution**
   - Navigate to the cloned folder
   - Double-click `Employee Management.sln` to open it in Visual Studio

3. **Build the project**
   - In Visual Studio, go to **Build → Build Solution** (or press `Ctrl + Shift + B`)

4. **Run the application**
   - Press **F5** or click **Start** to launch the app
